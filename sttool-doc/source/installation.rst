Installation
^^^^^^^^^^^^^^^
Dependencies
---------------
The ST-Tool is tested with the following dependencies and versions:

* ``Spark 3.0.0``
* ``Hadoop 3.2.1``
* ``Scala 2.12.10``
* ``Java 1.8.0``
* ``Python 3``

Add ``PATH/TO/STTOOL`` to environment variables:

E.g. for linux, add 

.. code-block :: json

    export PATH=$PATH: PATH/TO/STTOOL
to ``~/.bashrc``
